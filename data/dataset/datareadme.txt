Name of the dataset: complete_data

complete_data is the dataset we use for the analysis.
complete_data is a dataset compiled out of six different cleaned datasets.
These six datasets (in raw form) are downloaded from the links below:

Paris data:
1. http://data.insideairbnb.com/france/ile-de-france/paris/2021-12-07/data/listings.csv.gz
2. http://data.insideairbnb.com/france/ile-de-france/paris/2021-12-07/data/calendar.csv.gz

Rome data:
3. http://data.insideairbnb.com/italy/lazio/rome/2021-12-08/data/listings.csv.gz
4. http://data.insideairbnb.com/italy/lazio/rome/2021-12-08/data/calendar.csv.gz

Madrid data:
5. http://data.insideairbnb.com/spain/comunidad-de-madrid/madrid/2021-12-08/data/listings.csv.gz
6.http://data.insideairbnb.com/spain/comunidad-de-madrid/madrid/2021-12-08/data/calendar.csv.gz

----------------------------------------------------------

1. Motivation of data collection (why was the data collected?)

The data was collected to perform a statistical analysis. This analysis is performed to answer the following question: What is the effect of Valentine's Day on Airbnb listings in Europe? 

2. Composition of dataset (what's in the data?)

In the dataset is data from InsideAirbnb from three European cities. Namely the cities Paris, Rome and Madrid. 
The InsideAirbnb data contains listings data and calendar data.

3. Collection process (how was the data collected?)

The data is downloaded from the web. See src/data-preparation/download_data.R. 

4. Preprocessing/cleaning/labeling (how was the data cleaned, if at all?)

The preprocessing is all done in data-preparation stage. 

5. Uses (how is the dataset intended to be used?)

The dataset is intended to use for this research team to produce a workflow that answers the question "What is the effect of Valentine's Day on Airbnb listings in Europe?" for this team project.

6. Distribution (how will the dataset be made available to others?)

Raw data available for download for others. 

7. Maintenance (will the dataset be maintained? how? by whom?)

None.
